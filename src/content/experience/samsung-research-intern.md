- Engineered **BQ2Viz**, a multi-agent GenAI framework (LangGraph, LangChain, ChromaDB) that converts natural language into data visualizations, using a **Profiling Agent** for automated schema indexing and a **Self-Correcting Code Generator**.
- Designed a **Reasoning-Based Planner** for multi-file retrieval, inferring multiple distinct visualizations from a single abstract query (e.g., "Analyze Sales" → trend line + category bar + regional pie).
- Benchmarked against NVAGENT on the **VisEval** dataset: **84.57% accuracy** on complex multi-table queries, cutting the error rate to **11.5%** via a feedback-driven code-repair loop.
